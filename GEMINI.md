Font Styles
Element	Font Family	Weight	Size	Line Height
Headings	Inter or Poppins (sans‑serif)	600–700	24–32px	1.2–1.3
Subheadings	Inter	500–600	18–20px	1.3–1.4
Body Text	Inter	400	14–16px	1.5
Labels / Buttons	Inter	500	14px	1.4
Data / Shortlist	Roboto Mono or JetBrains Mono	400	13–14px	1.4
Rationale:

Inter is highly legible, open‑source, and works well on all screen sizes.

Poppins adds a slightly more modern, friendly feel for headings if desired.

A monospace font for scores and IDs ensures alignment and a technical, data‑focused look.

Color Palette
Role	Color Name	Hex Code	Usage
Primary	Deep Blue	#1E3A8A	Buttons, key actions, active states, accents
Primary Light	Bright Blue	#3B82F6	Hover states, secondary CTAs, highlights
Secondary	Teal	#14B8A6	Success messages, positive indicators, progress
Accent	Amber	#F59E0B	Warnings, attention, “gaps/risks” indicators
Neutral	Slate Gray	#1E293B	Headings, primary text
Neutral Light	Cool Gray	#F1F5F9	Backgrounds, cards, input fields
Text	Dark Gray	#0F172A	Body text, labels
Text Light	Medium Gray	#475569	Placeholders, secondary info
Error	Red	#EF4444	Validation errors, delete actions
Success	Green	#10B981	Confirmation, completed uploads
Rationale:

Deep Blue conveys trust and professionalism—important for HR tools.

Teal and Amber provide clear visual cues for candidate strengths vs. gaps.

Neutral grays keep the interface calm and focus attention on AI‑generated insights.

High contrast meets accessibility standards (WCAG AA), ensuring readability.

Visual Elements
Cards: Light gray background (#F1F5F9) with subtle border radius (12px) and shadow.

Shortlist Ranks: Use a circular badge with rank number (e.g., #1E3A8A background, white text).

Score Indicators: Progress bars or circular meters using primary or secondary colors.

Reasoning Panels: Light background with a vertical accent stripe in amber/teal to highlight strengths/gaps.
